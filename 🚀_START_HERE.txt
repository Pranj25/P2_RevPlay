2. Enter password when asked: PranjalP@2003

🚀 START APPLICATION (Every time):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

EASIEST WAY (NO ECLIPSE NEEDED):
  → Double-click: START_BOTH_SERVERS.bat
  → Wait 35 seconds
  → Browser opens automatically!

ALTERNATIVE (Interactive Menu):
  → Double-click: QUICK_START.bat
  → Choose option [2] Start Both Servers


📱 ACCESS APPLICATION:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Frontend:  http://localhost:4200  ← Main app (use this!)
Backend:   http://localhost:8081  ← API only
Swagger:   http://localhost:8081/swagger-ui.html  ← API docs


⚠️ IMPORTANT:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ NO ECLIPSE REQUIRED - runs from Command Prompt!
